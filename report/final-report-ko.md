## 4. 데이터셋 설계

| 항목 | 내용 |
|---|---|
| 출처 | SQuAD의 `context` |
| 형식 | 원천 JSON → JSONL messages |
| 샘플 수 | 실험 후 기록 |
| 정제 방법 | context 중복 제거, 길이 필터링, 새 질문 생성, 품질 검수 |

SQuAD의 기존 `question`과 `answers`는 직접 학습 대상으로 사용하지 않았다. `context`만 원천
텍스트로 사용하고, 프로젝트 목적에 맞는 새로운 질문을 생성하여 학습 데이터를 구축하였다.
